#### 切换权限
```
sudo su
```
#### 备份
```bash
cp -rf /etc/apt/sources.list /etc/apt/sources.list.save
```
#### 删除原来的
```bash
rm /etc/apt/sources.list
```
#### 新建
```bash
vi /etc/apt/sources.list
```
