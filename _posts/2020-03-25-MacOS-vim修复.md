#### 问题
使用vim时不能正常使用，报错
```
dyld: Library not loaded: /usr/local/opt/ruby/lib/libruby.2.3.0.dylib
  Referenced from: /usr/local/bin/vim
  Reason: image not found
Trace/BPT trap: 5
```
#### 解决
```
brew reinstall ruby vim
```
